# DALY-USB
iobroker adapter to interface with DALY BMS via serial/USB


Features
[] interface with a DALY BMS via USB-UART adater
[] installed as a iobroker adapter
[] read all available states according to official DALY BMS api
[] create read only states with correct units as kWh and V
[] create writable states for any writable api call of daly bms
[] light on CPU and RAM usage
[] autoamtically adapts to BMS configuration as "number of cells"
